# Michis Home Assistant Addon Repo

Add https://github.com/mrschloemp/ntfy-addon-HomeAssistant to repositories in Addon Store.

- NTFY-Home-Assistant-Addon
Dieses Addon installiert einen NTFY-Docker Container in Home Assistant.
Über die Addon Konfiguration werden Benutzer eingetragen und das Web-UI deaktiviert.
Somit kann der NTFY Server auch mit Cloudflair Tunnel Online Benachrichtigungen senden.
